# Cloud Engineering & Software Architecture Knowledge Hub

A structured repository dedicated to mastering cloud-native engineering, infrastructure optimization, and core software design principles. This space serves as a central hub for deep-dive technical notes, roadmap tracking, and collaborative study documentation.

---

## 🗺️ Learning Roadmaps

The repository is structured to separate system infrastructure, automation practices, and core computer science concepts into logical directories.

### 🏛️ Cloud & Devops Engineering

#### 1. Kubernetes Mastering (`/Phases_of_Kubernetes`)
A sequential, phased learning track covering container orchestration from foundations to advanced production-grade operations:
* **Phase 1:** Kubernetes Fundamentals (`phase1_kubernetes_fundamentals.md`)
* **Phase 2:** Deployments & Scaling (`phase2_deployments_scaling.md`)
* **Phase 3:** Services & Networking (`phase3_services_networking.md`)
* **Phase 4:** Configuration & Secrets Management (`phase4_config_secrets.md`)
* **Phase 5:** Storage & Persistence (`phase5_storage_persistence.md`)
* **Phase 6:** Health, Security, & Debugging (`phase6_health_security_debugging.md`)
* **Phase 10:** Custom Resource Definitions (`phase10_crds.md`)
* **Phase 11:** Operators Pattern (`phase11_operators.md`)
* **Phase 12:** GitOps Workflow via ArgoCD (`phase12_gitops_argocd.md`)
* **Phase 13:** System Observability (`phase13_observability.md`)
* **Phase 14:** CKA Certification Preparation (`phase14_cka_certification.md`)

#### 2. Containerization (`/Docker`) *(Upcoming)*
* Dockerfile optimization, multi-stage builds, layers management, container networking, and security scanning (Trivy).

#### 3. Infrastructure as Code (`/Terraform`) *(Upcoming)*
* Provider configurations, state management, modularization, workspaces, and policy-as-code implementations.

### 💻 Computer Science Fundamentals

#### 1. Data Structures & Algorithms (`/DSA`) *(Upcoming)*
* Asymptotic analysis, core data structures, algorithmic paradigms (sorting, searching, dynamic programming), and practical problem-solving implementations.
